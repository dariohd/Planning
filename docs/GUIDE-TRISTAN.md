## Important à savoir

Cette application est **autonome** : elle ne se connecte pas à l'ancien tableur Google. Les plannings affichés pour 2026 ont été **générés automatiquement** (structure de base : quarts, jours fériés). Tu complètes et corriges les présences **au fil de l'eau** (CP, maladie, absences, etc.).

---

## Vue Équipe (quotidien)

1. Filtre **équipe** en haut à gauche : sélectionne ton périmètre REAP.
2. Bascule **Mois** / **Semaine** selon le besoin.
3. **Clique sur une cellule** pour modifier le statut d'un collaborateur un jour donné.
4. **Modification groupée** : applique un statut à plusieurs personnes sur une plage de dates.
5. **Imprimer** : disponible en vue semaine.

### Statuts courants

| Code | Signification |
|------|----------------|
| M / A / N / J | Matin, Après-midi, Nuit, Jour |
| CP | Congés payés |
| JRTT | RTT |
| Ma | Maladie |
| Abs | Absence |
| F | Formation |
| P | Présent |

---

## Vue Individuelle

1. Sélectionne un collaborateur dans la liste à gauche.
2. Calendrier mensuel : clique sur un jour pour éditer.
3. **Compteurs** en haut : CP, JRTT, maladie, formation, etc.
4. **Appliquer une plage** : saisie rapide sur plusieurs jours (ex. une semaine de CP).

## Ce que tu ne peux pas faire (rôle REAP)

- Paramètres administrateur (accès utilisateurs, génération année globale, exports massifs).
- Modifier les présences hors de ton périmètre équipe.

Pour une demande admin, contacte Dario.

---

## Bonnes pratiques

1. Vérifie le filtre équipe avant toute modification groupée.
2. En cas d'erreur, utilise **Annuler** dans le bandeau notification (12 secondes).
3. Pour une semaine de congés : vue Individuelle > Appliquer une plage.
